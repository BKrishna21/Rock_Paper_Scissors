## Rock Paper Scissors Game

A simple and interactive **Rock Paper Scissors** game built using **HTML**, **CSS**, and **JavaScript**. The game allows the user to play against the computer, with scores updating dynamically after each round.

This project was created to practice JavaScript fundamentals, DOM manipulation, event handling, and basic game logic.

## 🎮 Features

* Play Rock, Paper, or Scissors by clicking on an option
* Computer generates a random move
* Instant win, lose, or draw result
* Live score tracking for both the player and the computer
* Dynamic messages with color changes based on the game outcome
* Clean and responsive user interface

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## 🎯 How to Play

1. Open the webpage in your browser.
2. Click on **Rock**, **Paper**, or **Scissors**.
3. The computer randomly selects its move.
4. The game displays the result:

   * 🟢 Win
   * 🔴 Lose
   * 🔵 Draw
5. Scores are updated automatically after each round.
